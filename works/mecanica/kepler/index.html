Kepler's Laws

Tomás Basile

Problem description

We want to figure out the orbits followed by planets when going around the sun. For that, suppose the sun has a mass \( M \) and located at the origin of coordinates. And we have a planet with mass \(m\) whose position we can describe using polar coordinates. We will suppose that the sun is kept still.

Because the sun is at the center of our coordinates, the gravitational force felt by the Earth is given by:

\( \vec{F} = - \dfrac{GMm}{r^2} \widehat{r} \)
Then, we use Newton's second law \( \vec{F} = m \vec{a} \) using the expression for acceleration written in polar coordinates, and therefore we get:
\( (\ddot{r} - r \dot{\theta}^2 ) \widehat{r} + (r \ddot{\theta} + 2 \dot{r} \dot{\theta}) \widehat{\theta} = - \dfrac{GM}{r^2} \widehat{r} \)
Which gives us two coupled differential equations:
  • \( \ddot{r} - r \dot{\theta}^2 = - \dfrac{GM}{r^2} \)
  • \( r\ddot{\theta} + 2 \dot{r}\dot{\theta} = 0 \)
It is easy to see that the second equation can be written as \(\dfrac{1}{r} \dfrac{d}{dt} (r^2 \dot{\theta}) =0\). So this implies that \(r^2 \dot{\theta} \) is constant. We usually multiply it by \(m\), which is also constant, and so we define:
\( L := mr^2 \dot{\theta} \)

Which we call the Angular momentum and it is a constant throughout the orbit.

Nest, we have that \( \dot{\theta} = \dfrac{L}{mr^2}\), which we can substitute into the first equation to get: \( \ddot{r} - \dfrac{L^2}{m^2r^3} = - \dfrac{GM}{r^2}\). We can multiply this equation by \(\dot{r}\) and then integrate with respect to time:
\( \int \ddot{r} \dot{r} dt - \int \dfrac{L^2}{m^2 r^3} \dot{r} dt = - \int \dfrac{GM}{r^2} \dot{r} dt \)

\(\Rightarrow \;\;\; \dfrac{\dot{r}^2}{2} + \dfrac{L^2}{2 m^2 r^2} = \dfrac{GM}{r} + E\)
Where \(E\) is an integration constant, we can simplify it to:
\(\dfrac{1}{2} m\dot{r}^2 + \dfrac{L^2}{2mr^2} - \dfrac{GMm}{r} = E \)
This is a very interesting equation which gives a qualitative understanding of the orbits. We can see that there is no dependance on \(\theta\) any more. Also, the first term looks exactly like the kinetic energy for a particle moving in one dimension with coordinate \(r\).
Meanwhile, the right side is constant number \(E\). So this kind of looks like an equation of conservation of energy. We already have the kinetic energy in the first term, so maybe we could think of the other two terms as making some kind of potential energy for the system, which we will call effective potential \(V_{eff}\). Therefore:

\( \dfrac{1}{2} m \dot{r}^2 + V_{eff}(r) = E \;\;\;\;\;\;\; (1)\)
Which looks exactly like a conservation of energy equation. The real potential for the system is the gravitational potential \(V = - \dfrac{GMm}{r} \). But this new effective potential allows us to write the equation with only the variable \(r\) and in such a way that we get the typical conservation of energy \(KE + V = E\).

Now, we will try to plot the effective potential \(V_{eff} = \dfrac{L^2}{2mr^2} - \dfrac{GMm}{r} \)

This function is plotted for some arbitrary values of \(L,M,m\) just to see the general form.
We see that the function goes to infinity when \(r\) tends to 0, then it has a minimum with a value below zero and it then tends to \(0\) as \(r\) increases.

The interesting thing is that equation (1) says tat \( \dfrac{1}{2} m\dot{r}^2 + V_{eff}\) is equal to a constant \(E\) which is the total energy of the system. But \(\dfrac{1/2} m \dot{r}^2\) needs to always be a positive number, so we must always have that \(E > V_{eff}(r)\) throughout the orbit.

This implies that given a (constant) value of \(E\), this determines the possible values of \(r\) for the orbit, which will be those for which \(E > V_{eff}(r)\).

This will give us a couple of different possibilities for the orbits depending on the value of \(E\):

  • E < \(V_{min}\): Where \(V_{min}\) is the minimum value taken by the effective potential. This case is impossible, because there will be no value of \(r\) such that \(E > V_{eff}(r)\). Therefore, there is no way of satisfying equation (1)
  • E = \(V_{min}\):
    • As we said before, the only possible values for \(r\) are those for which \(V_{eff}(r) < E\). In this cases, because \(E\) is equal to the minimum of \(V_{eff}\), there is just one valid value of \(r\), which we may call \(r_{min}\). Therefore, the orbit is at all times at some fixed distance from the sun, so it must be a circle.
  • 0>E>\(V_{min}\):
    • In this case we have plotted the function \(V_{eff}(r)\) as well as the constant value \(E\). As we said before, the possible values of \(r\) that the planet will have while going throught its orbit will be those for which \(V_{eff}(r) < E\). So in this case, there is a whole section of possible values of \(r\) to take, limited by the values \(r_{1}\) and \(r_2\). Therefore, the orbit will be some kind of curve that at a moment gets to a closest distance \(r_1\) to the sun and at another moment is at the farthest distance \(r_2\).
  • E \(\geq\) 0:
    • In this case, the values of \(r\) such that \(V_{eff}(r) < E\) has a minimum value as before, but includes arbitrary large values. So the planet can go infinitely far from the sun in its orbit. This is what we would call an unbounded orbit. The planet would not be "orbiting" in the literal meaning of the word, it would come close to the sun once, and then go infinitely far and never return.
Therefore, we can have a qualitative understanding of the form of the orbits with this analysis, though we don't know the exact formulas for the orbits yet.

Equations of the Orbits:

Now we will try to solve for the equation of the orbit. To do that we begin with the two equations we have:
\( mr^2 \dot{\theta} = L \) \( \dfrac{1}{2} m \dot{r}^2 + \dfrac{L^2}{2mr^2} - \dfrac{GMm}{r} = E\)
The procedure to find the orbit equation \(r(\theta)\) is long but generally straightforward, so I won't do it completely. The first step would be to isolate \(\dot{r}\) in the second equation and then divide both sides by \(L^2\), with that, we get:
\( \left( \dfrac{1}{r^2} \dfrac{dr}{d\theta} \right)^2 = \dfrac{2mE}{L^2} - \dfrac{1}{r^2} - \dfrac{2mV(r)}{L^2} \)
Finally, we solve for \(\dfrac{dr}{d\theta}\) in this equation and then integrate in order to get \(r(\theta)\) after many steps. The solution turns out to be:
\( r(\theta)= \dfrac{L^2}{m\alpha(1+\epsilon \cos \theta)} \)
Where we have defined \(\epsilon := \sqrt{1+ \dfrac{2EL^2}{m\alpha^2}}\) and \(\alpha := GMm\).
This result gives us the orbit of the planet in polar equations, giving us the value of \(r\) for any \(\theta\). The incredible thing is that this equation can be shown to be the polar representation of a conic section with eccentricity \(\epsilon\). So we have shown that planets always move in orbits with the forms of conic sections. The conic section in particular depends on the value of \(\epsilon\).

It is easy to prove using standard calculus, that the minimum value of the effective potential is \(V_{eff} = - \dfrac{\alpha^2}{2L^2}\)

Therefore, the eccentricity can be writen as:

\(\epsilon = \sqrt{1 - \dfrac{E}{V_{min}}} \)

Which explains the cases for the orbits we found earlier. If \(E = V_{min}\), the eccentricity is \(\epsilon = \sqrt{0} = 0\), which means that the orbit is a circle, just as we showed quantitatively earlier.

If \(E < V_{min}\), we have the square root of a negative number, and the solution makes no sense, as we had concluded earlier. If \( V_{min} < E < 0\), then \(0 < \epsilon < 1\), which mwans that the orbit is an ellipse. This coincides with the behaviour we described earlier, the orbit has a closest and a farthest point.

Finally, when \(E=0\), then \(\epsilon = 1\), so the orbit is a Parabola and if \(E >0\), then \(\epsilon > 1\) and the orbit is a hyperbola. In either case, the planet only gets close to the sun once and then goes to infinity, just as we described earlier.

Then press start to see what orbit the planet makes. The program divides the motion in many little steps, and in each one of them it calculates the gravitational force and accelerates the planet accordingly. The program doesn't know what the orbit will look like, it just draws step by step according to the acceleration given by gravity. Still, you can see that the results are conic orbits, which assures us once again, now computationally, that the gravitational force leads to conic orbits.



Try varying the initial speed and notice how for high speeds (which have high energy), the orbits are usually unbounded.

The fact that gravitational orbits are conic sections is known as Kepler's first law.
