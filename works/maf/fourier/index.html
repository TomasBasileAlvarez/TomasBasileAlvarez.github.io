What is a Fourier Series?

Putting it simple, a Fourier Series is a way of writing a function as a sum of sines and cosines.
For example, suppose we have a function \( f \) defined over an interval \( [-L,L] \). The fourier series of this function is a way of writting \(f(x)\) on this interval as a sum of cosines and sines of different frequencies. That is, we can write \(f(x)\) as:
\(f(x) = \dfrac{a_0}{2} + \sum\limits_{n=1}^{\infty} \left[a_n \cos \left( \dfrac{n\pi x}{l}\right) + b_n \sin \left( \dfrac{n\pi x}{l} \right) \right] \)
Where \(a_n\) and \(b_n\) are real numbers, whose values depend on the function \(f(x)\).
In some sense, this means that the set of functions \({ \cos \left( \dfrac{\pi x}{l}\right),\sin \left( \dfrac{\pi x}{l}\right),\cos \left( \dfrac{2\pi x}{l}\right),\sin \left( \dfrac{2\pi x}{l}\right),\cos \left( \dfrac{3\pi x}{l}\right),\sin \left( \dfrac{3\pi x}{l}\right), \cdots \}\) is like a basis for all functions defined on the interval \([-L,L]\).
