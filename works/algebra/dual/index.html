Dual Space

Tomás Basile

Definition

We start by considering any vector space \(V\) over a field \(F\) (which could be \(R\) or \(C\)). Then the dual space, denoted by \(V^*\) is defined as the set of all linear maps \(\phi : V \rightarrow F\), that is:
\( V^* = \{\phi \; | \; \phi: V \rightarrow F \; \text{is linear} \}\)
This dual space is itself a vector space over the same field \( F\) when we define it an adittion and scalar multiplication:
  • \( (\phi + \psi)(x) = \phi(x) + \psi(x) \)
  • \((a \phi) (x) = a (\phi(x))\)
For \(\phi , \psi \in V^* \; ,\; x \in V \; ,\; a \in F \).

Basis

If we are in a finite dimensional vector space, such that the basis of \(V\) is given by \(\{e_1 , e_2 , \cdots , e_n\} \), we can construct a basis for \(V^*\) quite simply.

To do it, we define a set \( \{ e^1 , \cdots , e^n \} \)of elements of \(V^*\) defined by:

\( e^i(e_j) = \delta_{j}^i\)
That is, the element \(e^i\) of \(V^*\) is a functional defined in such a way that when we apply it to the ith element of the basis of \(V\) the result is 1, and when we apply it to any other element of the basis of \(V\), the result is 0.

It is easy to convince oneself and prove that this is really a basis for \(V^*\).

So \(V\) and \(V^*\) have the same dimension and given a basis of \(V\), we can find a basis of \(V^*\).

Bi Dual Space

Now we will consider the space \( (V^*)^*\) which is the dual space of \(V^*\). According to our definition of dual, this space is defined as:

\( (V^*)^* = \{ \alpha \; | \; \alpha: V^* \rightarrow F \;\; \text{is linear} \} \)
That is, \( (V^*)^* \) is the set of all linear functions that take an element of \(V^*\) and output an element of the field \(F\).

It can be a little bit difficult to wrap one's head around this space, but there is an amazing correspondance between \( (V^*)^* \) and \(V\) that makes it way easier.

Consider any vector \(x\) living in \(V\), then, we can find a vector \(\bar{x}\) in \((V^*)^* \) defined as:

\( \bar{x}(\phi) := \phi(x) \)

For any \( \phi \in V^* \). We define the action of \(\bar{x}\) on \(\phi \in V^*\) as being given by \(\phi(x)\), which is an element of \(F\). It is easy to see that this \( \bar{x}\) is truly an element of \((V^*)^*\) because it is also linear: \( \widehat{x} (\phi + \psi) := (\phi + \psi)(x) = \phi(x) + \psi(x) = \bar{x}(\phi) + \bar{x}(\psi) \).

Therefore, for any element of \(V\) we can find a corresponding element of \((V^*)^*\). Furthermore, because dual spaces have the same dimension as the original spaces, \((V^*)^*\) has the same dimension as \(V\). So we can conclude that this correspondance is biyective. This is an amazing fact, that there exists a very natural correspondance between \(V\) and \((V^*)^*\) that doesn't depend on basis representation or anything else.
