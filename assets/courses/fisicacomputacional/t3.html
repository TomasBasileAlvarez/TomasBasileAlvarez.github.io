Primero definimos una funci√≥n polL que toma como entrada n√∫meros n,k, el conjunto de puntos xp en los que vamos a interpolar y la variable x.

PolL da como resultado la funci√≥n $L_k$ evaluada en el punto x. Donde $L_k$ es el k-√©simo polinomio base como lo definimos en clase, que dados los puntos $[x_0, x_1, \cdots, x_n]$ sobre los que vamos a interpolar se define como:

$$ L_k(x) = \frac{(x - x_0) \dots (x - x_{k-1})(x - x_{k+1}) \dots (x - x_n)}{(x_k - x_0) \dots (x_k - x_{k-1})(x_k - x_{k+1}) \dots (x_k - x_n)}$$

def polL(n,k,xp,x):
    # res va a ser el resultado de L_k(x)
    # Empezamos con res =1
    res = 1 
    
    # La funci√≥n Lk(x) consiste en multiplicar n+1 factores de la forma
    # (x-x_i)/(x_k-x_i)  excepto para i=k
    # As√≠ que vamos a iterar la i desde 0 hasta n+1
    for i in range(n+1):
        # si i es distinto de k
        if i!=k:
            #multiplicamos res por el i-√©simo factor
            res*=(x-xp[i])/(xp[k]-xp[i])
    # Al terminar los productos, ya tendremos la soluci√≥n
    return(res)

Ahora definimos una funci√≥n Lagrange(n,xp,yp) que ya construye el polinomio de lagrange de orden n. Donde xp es la lista de los puntos muestra en el eje x y yp son los valores de la funci√≥n que queremos interpolar en dichos puntos.

Como vimos en clase, la soluci√≥n est√° dada por $$ P_n(x) = y_0 L_0(x) + y_1 L_1(x) + \dots + y_n L_n(x) $$

La funci√≥n va a devolver el polinomio de lagrange en forma simb√≥lica usando sympy

# definimos la funci√≥n
def lagrange(n,xp,yp):
    # len(xp) es la cantidad de puntos muestra que tenemos
    # El polinomio de lagrange de mayor grado que podemos contruir con len(xp) puntos
    # es de grado len(xp)-1. As√≠ que si el usuario quiere un polinomio de grado
    # mayor a este n√∫mero, autom√°ticamente convertiremos n en este m√°ximo posible
    if n>len(xp)-1:
        n = len(xp)-1

    # Empezamos con un resultado de 0
    res = 0
    
    
    
    # vamos a ir sumando los t√©rminos y_i L_i(x) 
    # donde L_i(x) lo conseguimos con la funci√≥n polL(x)
    # si n+1 es menor a la cantidad de puntos que hay en la muestra,
    #lo que pasar√° es que s√≥lo consideraremos los primeros n+1 puntos muestra
    # y los dem√°s ser√°n ignorados
    for i in range(n+1):
        # la variable x la hab√≠amos importado en from sympy.abc import x
        # por lo que es un s√≠mbolo de sympy y se va a mantener simb√≥lico
        res += yp[i]*polL(n,i,xp,x)
        
    # La funci√≥n expand expande este polinomio para escribirlo como
    # a_nx^n + a_{n-1}x^{n-1} + ... + a_1 x + a_0
    return(expand(res))

from IPython import display
import time
import random

#creamos una serie de puntos entre -pi y pi
xr = np.linspace(-np.pi,np.pi)

# definimos una funci√≥n que grafica puntos muestra, el polinomio y el seno
def visual(Xp,Yp,p):
    plt.plot(xr, p(xr), 'k', label='y = p{}(x)$'.format(len(Xp)-1))
    plt.plot(Xp, Yp, "o")
    plt.xlim(-np.pi,np.pi)
    plt.ylim(-1,1)
    plt.plot(xr,np.sin(xr),'--')
    display.clear_output(wait=True)

    plt.show()
    

# vamos a ir agregando puntos hasta tener maxx puntos
maxx = 10

plt.figure(figsize=(12, 5))

#empezamos con un solo punto -pi
Xp = [-np.pi]

# vamos a ir agregando puntos y cada vez vamos actualizando el polinomio
# y lo graficamos de nuevo
for i in range(2,maxx+1):
    # agregamos puntos alternativamente al inicio y final del intervalo [-pi,pi]
    if i%2 == 0:
        Xp.append(-np.pi + 2*np.pi/maxx*(i//2))
    else:
        Xp.append(np.pi - 2*np.pi/maxx*(i//2))
    
    Yp = np.sin(Xp)
    
    #creamos el polinomio de lagrange
    p = lagrange(i,Xp,Yp)
    
    #graficamos
    visual(Xp,Yp,lambdify(x,p))
    # esperamos antes de actualizar la gr√°fica
    time.sleep(0.6)

    return(2*np.sin(np.pi/6 * x))
