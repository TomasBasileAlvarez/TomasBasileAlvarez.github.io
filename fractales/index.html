Self Similar Fractals

Tomás Basile

Introduction

The concept of a fractal is pretty general in mathematics, so we will consider only a special type of fractals.

The types of fractals we will consider are what we will call self similar fractals. That is, as a first definition, sets that have subsets that are similar to the complete set. Moreover, we will concentrate only on fractals defined inside $\R^2$.

To understand this, we will need some definitions first

Similarity Transformation

We will define a similarity transformation on a set $V$ as a function $f: R^2 \rightarrow R^2$ that 'preserves fomrs'. That is, for any subset of $R^2$ on which we apply the transformation, the result is similar to $R^2$

We will only care for transformations conformed of a composition of rotations around the origin, translations and homogeneous scaling (multiplying all the set by a scalar)
This transformations clearly 'preserve forms'.

If a set $B$ can be obtained by applying a similarity transformation to a set $A$, we say that $A$ and $B$ are similar sets (intuitively, $A$ and $B$ have the same form, only rotated, translated and scaled).

Self similar set

We will define a self similar set (what we will call a fractal) as a set $E \subset \R^2$ such that there exists a set of similary transformations $f_1, f_2, \cdots , f_n : R^2 \rightarrow R^2$ with:

  1. $f_i(E) \subset E \;\;\;\; \forall i$
  2. $E = f_1(E) \cup f_2(E) \cup f_3(E) \cdots \cup f_n(E)$

The first condition tells us that applying any of the functions to the whole set E, gives us as a result a part of E. Since the functions are similarity transformations, this encapsulates the intuitive definition of a fractal, that a part of it is similar to the whole set.

The second condition just tells us that the ser $E$ can be constructed from applying every function to $E$ and joining the results. That is, the set $E$ is formed by n parts (each one beign $f_i(E)$) that are each similar to the whole set.

To understand this better, we can see the following example:

This famous set is called the Sierpinski triangle and it is the quintessential example of a self similar fractal.

It is easy to see that the set is divided in three subsets that are similar to the triangle as a whole (the red, blue and green subtriangles). This means that each of these three subtriangles can be obtained by applying a similarity transformation to the whole triangle.

That is, there is a similarity transformation $f_1$ that when applied to the whole triangle gives as a result the red subtriangle. And there is another function $f_2$ that when applied to the whole triangle gives as a result the blue subtriangle, and a function $f_3$ that gives as a result the green one.

In this case, the functions $f_1, f_2 , f_3$ consist only in scaling the whole triangle by 1/2 and then translating it in different directions for each function.

That way, we could say that the triangle is composed of three subparts that are similar to the whole set

We can see that this set with this three functions satisfies the definition we gave for a fractal.
