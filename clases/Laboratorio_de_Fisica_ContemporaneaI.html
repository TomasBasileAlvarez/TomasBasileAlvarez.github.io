Laboratorio de Física Contemporánea I



Este curso es un laboratorio avanzado, en el que se realizan prácticas más largas que en los otros labortarios y de temas contemporáneos de la física (mecánica cuántica, materia condensada, física nuclear, etc.).
Las prácticas que realicé fueron las siguientes:

  1. Medición de la Constante de Stefan Boltzman: En esta práctica se utiliza la ley de Joule para estimar la radiación emitida por un alambre de cobre por el que pasa electricidad. Luego se combina esto con una aproximación de la temperatura a la que se encuentra el cobre para estimar la constante de proporcionalidad que relaciona a la potencia de la radiación emitida con la temperatura (conocida como constante de Stefan Boltzman).

  2. Ley de Bragg: En esta práctica se usa la ley de Bragg para calcular la distancia interplanar de un arreglo cristalino a partir de la difracción de ondas electromagnéticas sobre él. Además, se escribió el código en python para poder hacer una simulación computacional del experimento. La forma en que funciona el código se explica en la práctica y anexo el código aquí abajo.

  3. Análisis de Datos (Tipo de cambio USD/MXN): Esta práctica es muy distinta a las demás, pues no es de física, sino de análisis de datos. En este trabajo se buscan variables económicas y sociales que puedan estar relacionadas con el tipo de cambio entre el dólar y elpeso mexicano (como el precio del petróleo, las remesas, el desempleo, etc.). Se descargaron estos datos, y se limpiaron, para después medir la correlación entre cada dato con el tipo de cambio USD/MXN.
